indian-e-visa.org
This site was built with angular 4 on the front end with bootstrap. It has automated email functionality to contact customers as well as stripe integration for payments. The main functionality is a form submission for the application of the visa and payment via credit card for the amount corresponding to the user's naitonality. There is also a customer management system i built for administrators with access rights which is is not accessible to users.




My cms

This is a content mangement system i built for myself. It's a basically a way to save data to a db to make up for my bad memory. I've been using a system like this for about a year and it's gone through a few iterations. It was built on angularjs at first but i have since migrated to ngx. I host it on heroku behind some authentication with amazon's s3 service for file and document uploads. I use it to track my lifting progress and a dream journal, my blog is also updated through this. This system also has admin rights to some websites i have built. For obvious reasons i can't provide a link and have blacked out some bits.
